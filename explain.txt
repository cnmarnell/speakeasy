transcript:
The transcript for the audio segment being processed.


confidence:
A floating point value between 0 and 1 that indicates overall transcript reliability. Larger values indicate higher confidence.


words:
An array containing each word in the transcript, along with its start time and end time (in seconds) from the beginning of the audio stream, and a confidence value.

Including smart_format=True in listen.prerecorded('1').transcribe_url adds a few fields to the response:
each word object also includes its punctuated_word value, which contains the transformed word after punctuation and capitalization are applied.
each alternatives object also includes a paragraphs object. This object also contains sentences for each paragraph, along with a count of the num_words in the paragraph, and the start and end times for each paragraph.
